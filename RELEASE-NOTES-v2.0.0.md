# Version 2.0.0

New multi-store commerce tools:

- Price Manager: percentage, fixed, and exact price operations with preview.
- Inventory Manager: increase, decrease, and set quantities by location.
- Customer Import and Export: CSV, XLSX, and JSON.
- Order Import and Export: line-item exports and historical order creation.
- Blog Import and Export: blogs, articles, HTML, images, and SEO.
- Discount Manager: automatic and code discounts by vendor, type, tag, collection, category, or query.
- SKU Product Image Manager: download and upload SKU_01, SKU_02 ZIP archives.
- Store Media Export: Shopify Files, collection covers, and article images.

The Windows and macOS installer workflow remains in `.github/workflows/build-installers.yml`.
